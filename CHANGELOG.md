# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.1] - 2026-01-26

### Added
- Memo Replies: Support for a second level of memos (threaded conversations)
- Reply button in memo action tooltip

### Changed
- Renamed "Task Indentation" setting to "Markdown Indentation" (applies to both tasks and memos)
- Optimized "Open report" button logic to prevent opening duplicate tabs and instead focus on an existing tab if the report is already open

## [1.1.3] - 2026-01-26

### Added
- Task indentation style setting - choose between tab, 2 spaces, or 4 spaces

### Changed
- Task parser now supports mixed indentation styles (tabs and spaces)

### Fixed
- Schedule strikethrough not updating after accepting event in calendar

## [1.1.2] - Previous Release

_Initial public release with core features._
